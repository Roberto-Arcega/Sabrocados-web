# Estructura Landing Page - Sabrocados

## Secciones esenciales

| Sección | Propósito |
|---------|-----------|
| **Hero** | Primera impresión + CTA principal |
| **Productos** | Sabores/presentaciones disponibles |
| **Nutrición** | Info nutricional (lo que ya tienes) |
| **Experiencias** | Casos de uso: camping, gym, reuniones |
| **Testimonios** | Prueba social |
| **CTA Final / Contacto** | Cerrar la venta |

## Secciones opcionales (pero recomendadas)

| Sección | Por qué agregarla |
|---------|-------------------|
| **Cómo se hace** | Genera confianza, muestra calidad artesanal |
| **Comparativa** | "vs otros snacks" - destaca lo bajo en carbs, sin conservadores |
| **FAQ** | Resuelve dudas: ¿cuánto dura?, ¿cómo conservar?, ¿es keto? |
| **Dónde comprar** | Si vendes en tiendas físicas o marketplaces |

## Flujo recomendado

```
Hero
  ↓
Productos (qué vendes)
  ↓
Nutrición (por qué es bueno)
  ↓
Experiencias (para quién es)
  ↓
Cómo se hace (opcional - confianza)
  ↓
Testimonios (prueba social)
  ↓
FAQ (opcional - resolver objeciones)
  ↓
CTA Final / Contacto
```
